# Senior Care Robot

## Overview
This ROS2 package is developed for the Capstone project Senior Care Robot.  
It provides the basic structure for implementing the high-level control logic of a senior care robot.

---

## Operating Environment
- **ROS2 Humble**
- **Ubuntu 22.04**
